List Person records

query Parameters
name
string non-empty

Person's name (case-insensitive, similarity search)

surname
string non-empty

Person's surname (case-insensitive, similarity search)

patronymic
string

Part of Person's patronymic (case-insensitive, similarity search, empty for no patronymic)

age_min
integer (Age) [ 0 .. 125 ]
Example: age_min=42

Minimum for Person's age

age_max
integer (Age) [ 0 .. 125 ]
Example: age_max=42

Maximum for Person's age

nationality
string (CountryCode) = 2 characters ^[A-Z]{2}$
Example: nationality=RU

Person's nationality (ISO 3166-1 alpha-2 code)

sex
string (Sex)
Enum: "male" "female"
Example: sex=male

Person's sex

threshold
number [ 0 .. 1 ]
Default: 0

Threshold for similarity search (0.0 to 1.0)

offset
integer >= 0
Default: 0

The number of records to skip

limit
integer >= 0
Default: 20

The numbers of records to return (all if 0)
